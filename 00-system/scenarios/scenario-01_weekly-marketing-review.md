## 미션 1: 경쟁사 신호 + 자사 성과를 한 페이지로

윤정님의 첫 번째 요청. 대표님이 월요일 위클리에서 제일 먼저 물어볼 것.

> **윤정님**: 대표님이 먼저 물어보실 거예요
> "지난 주 시장에서 뭐가 떴고, 우리는 잘 하고 있어?" 이거요
>
> 그냥 표가 아니라 "헤드라인 3줄" 이 핵심이에요
> 그 위에 경쟁사 신호 카드 3~5장, 그 아래 자사 콘텐츠 분류
> 마지막에 다음 주 액션 3개로 끝나야 해요

**(속으로)**: 헤드라인 3줄에 다음 주 액션까지... 평소엔 한 줄 뽑는 데도 30분씩 걸렸는데. 이번에는 데이터부터 같이 봐주는 도구를 써보자.

**분석 요청**:
```
@competitors_master.csv @competitor_posts.csv @own_posts.csv @ad_campaigns.csv
@templates/joint-camp/weekly-marketing-review.md 양식대로
지난 1주(2026-05-08 ~ 2026-05-14) 위클리 마케팅 리뷰를 만들어줘.
헤드라인 3줄 → 경쟁사 신호 카드 3~5장 → 자사 성과 분류 → 광고 ROAS + 다음 주 액션 순서로.
경쟁사 서지(surge_flag=Y)는 우선 짚어주고, 자사 적용 가능성 한 줄씩 붙여줘.
```

**결과 확인 후 추가 질문 예시**:
```
"대표님 보고용으로 헤드라인 3줄을 다시 다듬어줘. 숫자 하나 이하, 35자 이내. 다음 주 액션이 자연스럽게 보이도록"
```

## 미션 2: 다음 주 광고 예산 재조정 — 멈출 캠페인 짚기

미션 1 결과를 받고 나니 동현님이 다시 슬랙을 보낸다.

> **동현님**: 반야님 빠르네요!
> 그 광고 ROAS 부분만 좀 더 자세히 봐 주실 수 있어요?
>
> 멈출 캠페인을 정확히 짚어야 다음 주 예산을 어디에 더 넣을지 정해요
> ROAS 1.0 미만 + 1.5 미만 두 단계로 나눠서 보여주시면 좋겠어요
> 채널별·국가별로도요

**(속으로)**: 미션 1의 광고 섹션을 그대로 확장하면 되겠다. 같은 데이터를 다른 각도로 보는 거니까.

**분석 요청**:
```
@ad_campaigns.csv
@templates/joint-camp/weekly-marketing-review.md 의 광고 비효율 검출 규칙 (ROAS < 1.0 / < 1.5) 적용해서
캠페인별 14일 평균 ROAS + 최근 7일 ROAS 같이 보여줘.
- ROAS < 1.0 캠페인 → 즉시 중단 후보 (이유 한 줄)
- ROAS 1.0~1.5 → 소재 교체 후보 (이유 한 줄)
- ROAS > 2.0 → 예산 증액 후보 (어디서 옮길지 한 줄)
채널 × 국가 매트릭스도 같이.
```

**결과 확인 후 추가 질문 예시**:
```
"중단 후보로 짚힌 캠페인의 광고 예산을 증액 후보 캠페인에 옮기면 다음 주 예산이 어떻게 재분배되는지 한 표로 만들어줘"
```
