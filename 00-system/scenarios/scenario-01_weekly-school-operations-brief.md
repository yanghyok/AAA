## 미션 1: 다음 주 5/20(수) 11개교 수업 운영 점검

은지님이 가장 먼저 보고 싶어할 것. 혜선님이 내일 부트캠프 끝나고 같이 보고 싶어할 것.

> **은지님**: 혜선님이 5/20 운영 한 번 같이 보자고 하셨어요
> 11개교 한꺼번에 가는 회차라
>
> 학교별로 차시·앙꼬·발송상태가
> 한 화면에 보이면 회의가 빠르게 풀려요
>
> 그리고 학교 이름은 풀네임으로
> 새빛고는 강남·송파 둘 다 있어서
> 풀네임 안 쓰면 한 번 더 물어보셔야 해요

**(속으로)**: 같은 날 11개교라는 게 처음엔 막막했는데, 표 한 장이면 회의 자리에서 막 부르고 짚어드릴 수 있겠다.

**분석 요청**:
```
@school_master.csv @class_schedule.csv
@templates/freewill/weekly-operations-brief.md 의 섹션 1 관점으로
2026-05-20(수) 수업 예정 11개교의 운영 점검표를 만들어줘.
학교명은 풀네임 유지, 차시 chip 형식(N차시/총차시), 발송상태 알림 자동 판정까지.
```

**결과 확인 후 추가 질문 예시**:
```
"혜선님 보고용으로 헤드라인 3줄 뽑아줘. 숫자 나열 말고 가장 우려되는 학교 한 곳을 콕 짚어서. 학교명은 풀네임으로"
```

## 미션 2: 직전 주 후기 미확인 이슈 — 본부 컨택 필요한 것

유나님과 지윤님이 같은 데이터에 다른 관심을 가졌다.

> **유나님**: 인스타 사례용은 이슈 없는 학교만 가져갈게요
> 미확인 이슈 있는 학교는 따로 정리해 주시면
> 저는 그 목록 빼고 콘텐츠 짤게요

> **지윤님**: 저도요. 이슈 본문 한 줄 요약만 있어도
> 어느 차시에서 시간 빠듯했는지 패턴 보이거든요

**(속으로)**: 같은 후기 데이터 한 번만 돌려서 둘 다 답해줄 수 있다. 굳이 따로 안 봐도 되겠다.

**분석 요청**:
```
@class_reviews.csv
@templates/freewill/weekly-operations-brief.md 의 섹션 3 관점으로
2026-05-04~2026-05-13 사이 작성된 후기 중 이슈여부=Y이면서 대응상태=미확인 인 건만 추려서,
심각도 순으로 정렬한 이슈 카드 목록을 만들어줘.
각 카드에 대응 제안 한 줄 포함.
```

**결과 확인 후 추가 질문 예시**:
```
"이 중에서 심각도 '심'은 본사가 학교 직접 컨택해야 하는 거잖아. 누구한테 어떻게 컨택하면 좋을지 학교별로 한 문장씩 제안해줘"
```
