## 미션 1: 작품별 진행 상황 — 가장 우려되는 한 곳

영주님이 먼저 물어볼 것. 부사장님이 페이지를 펴자마자 보고 싶어할 것.

> **영주님**: 부사장님이 먼저 물어보세요
> "오늘 가장 우려되는 작품 한 곳이 뭐야?" 이거예요
>
> 그러니까 페이지 최상단에 그 한 줄이 와야 해요
> 그 아래에 작품 8개 상태 점검표
> 같은 작품에 미완료 액션이 쌓여 있으면 별도 표시
>
> 위험·지연은 빨간색으로, owner 분명하게

**(속으로)**: 액션 아이템이 같은 작품에 누적되는 패턴을 빨리 잡아야 한다. 회의록 다 훑을 시간 없다.

**분석 요청**:
```
@projects_master.csv @meeting_actions.csv
@templates/joint-camp/project-tracker-brief.md 양식대로
작품 8개의 진행 상황 점검표를 만들어줘.
- 페이지 최상단 "오늘 가장 우려되는 한 곳" 한 줄 (액션 ID 인용)
- 작품 × 상태 × 우선순위 × 다음 마감 × 🔴🟡🟢
- 같은 project_id에 미완료 액션 3건 이상이면 별도 표시
- 지연(status=지연) 또는 due_date - 오늘 ≤ 3일은 🔴
```

**결과 확인 후 추가 질문 예시**:
```
"가장 우려되는 한 곳에 대해 부사장님께 드릴 한 줄 설명 + 후속 액션 owner·due_date 카드 한 장으로 정리해줘"
```

## 미션 2: 작품별 계약 조건 비교 — 표준화 어려운 지점 짚기

시우님 요청. 다음 주 임원회의 자료.

> **시우님**: 매니저님, 계약 조건이 작품마다 다 달라서 표준화가 어렵잖아요
> 부사장님께 드릴 비교 자료는 "지금 다른 게 어떤 지점인지"가 분명히 보여야 해요
>
> 정산 방식이 비율(60:40)인지 로열티(%)인지 혼재한 게 핵심이고요
> 협상 중인 건은 별도 표시 + 후속 액션이 뭔지도요

**(속으로)**: 계약 비교는 보통 엑셀에서 하루 종일 걸리는데... 액션 아이템과 묶어서 한 번에 볼 수 있으면 시우님도 편하다.

**분석 요청**:
```
@projects_master.csv @contract_terms.csv @meeting_actions.csv
@templates/joint-camp/project-tracker-brief.md 의 섹션 3 (계약 조건 비교 카드) 양식으로
작품별 권리·정산·홀드백·승인 조건을 한 표로 모아줘.
- 정산 방식이 비율(N:M)인지 로열티(%)인지 컬럼 분리해서 강조
- 협상 중·검토 중인 건은 별도 표시 + 관련 미완료 액션 ID 인용
- "표준화 어려운 지점" 한 줄 요약 (정산 방식 혼재 / 홀드백 편차 / 승인 요건 유무 각각)
```

**결과 확인 후 추가 질문 예시**:
```
"이 비교표 기준으로 향후 신규 계약 표준 양식 초안을 한 페이지로 제안해줘. 어느 조건은 고정값으로, 어느 조건은 작품별로 협상하면 좋을지"
```
